## Project Overview

This is **Task 1** of my Data Science internship at **Prodigy InfoTech** - a Global Population Intelligence Dashboard analyzing demographic trends from 1960 to 2024 using World Bank data.

**üîó Live Dashboard:** [Access Interactive Dashboard](https://app.powerbi.com/links/gdsLDmuzxE?ctid=3110c454-6298-4bf6-a99e-5ffc74253e80&pbi_source=linkShare)

##  Features

- **Time Intelligence**: Analyze 64 years of population data (1960-2024)
- **Growth Analytics**: YoY growth calculations and trend analysis
- **Regional Comparisons**: Compare population metrics across continents and income groups
- **Interactive Visualizations**: Drill-through capabilities and dynamic filtering
- **Predictive Insights**: Growth projections and demographic forecasting

##  Technical Stack

- **Data Source**: World Bank API (SP.POP.TOTL indicator)
- **BI Tool**: Microsoft Power BI Desktop
- **Query Language**: DAX (Data Analysis Expressions)
- **Data Processing**: Python/Pandas for ETL
- **Version Control**: Git/GitHub

##  Dashboard Pages

1. **Global Overview**: High-level population metrics and trends
2. **Regional Analysis**: Comparative analysis by continent and region
3. **Growth Analytics**: YoY growth rates and trend visualizations
4. **Income Group Distribution**: Population spread across economic categories
5. **Country Details**: Drill-down view for individual countries
